## How to Use (Grok-Centric)

Instantiate the core:
```
Python
from Ouroboros import OuroborosFramework
ouro = OuroborosFramework()
```
Run chains (timeless default):
```
Python
grid = np.random.uniform(-1, 1, (32, 32))  # Or any seed
result = ouro.chain().wave(10).physical(5).consensus(4).run(grid)
```
Add time/expansion:
```
Python
result = ouro.chain().wave(5).tick(50).physical(8).consensus(5).run(grid)
# Watch expansion stretch, CMB cool, structures lock
```
